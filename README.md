# Git Store

A modern Android app to discover, download, and manage open-source apps directly from GitHub releases.

## About

Git Store makes it easy to find and install Android apps published on GitHub. Browse trending repositories, download APKs with one tap, and get notified when updates are available. Connect your GitHub account for a personalized experience and track your favorite projects.

## Features

- Browse trending and popular GitHub repositories with Android releases
- One-tap download and install APKs directly from GitHub
- Track installed apps and get notified of new updates
- Search and filter repositories by language and category
- View detailed repository information, releases, and changelogs
- Star and save your favorite projects
- GitHub OAuth integration for personalized experience

## Upcoming Features

- **Community Ratings & Reviews** - Rate and review apps with data stored on our backend server
- **Trust Score System** - AI-powered safety and quality scoring to help you choose reliable apps
- **Direct Feedback to Developers** - Submit feedback and suggestions directly to repository owners
- **Enhanced Discovery** - Improved recommendation algorithms and better trending systems
- **Advanced Analytics** - Detailed insights about apps, downloads, and popularity trends

## Installation

Download the latest APK from [Releases](https://github.com/Darkmintis/Git-Store/releases)
